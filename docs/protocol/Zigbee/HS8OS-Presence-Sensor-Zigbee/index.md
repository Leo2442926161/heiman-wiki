# HS8OS - Occupancy Sensor (Zigbee)

## Overview

The HS8OS is Heiman's Zigbee occupancy sensor, employing millimeter-wave radar technology to detect stationary human presence (e.g., lying down, sitting still), overcoming the limitation of traditional PIR sensors which can only detect movement.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Millimeter-Wave Radar (mmWave) |
| Detection Range | Radius 3 ~ 5 m |
| Detection Capability | Stationary human presence (including micro-movement breathing detection) |
| Power Supply | AC Powered |
| Installation Method | Ceiling Mount |

## Features

- Stationary human detection (breathing micro-movement)
- Coverage radius 3~5 m
- Ceiling mount for optimal detection angle
- Can complement PIR sensors

## Application Scenarios

- Smart lighting: lights on when occupied, auto-off when vacant
- HVAC energy saving: auto-turn off AC when无人
- Smart elder care: alert on prolonged inactivity
