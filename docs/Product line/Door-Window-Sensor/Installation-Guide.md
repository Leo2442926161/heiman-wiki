# Installation Guide

## Mounting Location Requirements

- Main unit (transmitter) installed on **door/window frame**
- Magnet installed on **door/window leaf**
- Gap between main unit and magnet **≤ 10 mm**
- Align both, lining up the alignment marks

## Installation Steps

1. Clean mounting surface, ensure dry and dust-free
2. Peel off 3M adhesive protective film, attach main unit to door frame
3. Attach magnet at corresponding position (aligned with main unit when closed)
4. Insert battery (CR1632), LED flashes to indicate power on
5. Open/close door or window test: door open triggers alarm, closed restores
6. Pair via gateway

## Important Notes

- Excessive mounting gap will cause detection failure
- Metal doors/windows may affect wireless signal, recommend testing before finalizing
- Battery life ~2-3 years, APP will notify when low
