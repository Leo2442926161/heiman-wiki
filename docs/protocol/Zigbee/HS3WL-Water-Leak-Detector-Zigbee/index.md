# HS3WL - Water Leak Detector (Zigbee)

## Overview

The HS3WL is Heiman's 3rd generation Zigbee water leak detector, with enhanced detection sensitivity and wireless performance over the previous generation, suitable for water leak monitoring scenarios.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Detection Method | Probe Detection |
| Power Supply | DC 3V Battery |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH |

## Features

- Real-time water leak / accumulation detection
- Enhanced detection sensitivity
- Buzzer alarm
- Low battery reminder
- Probe anti-corrosion design
