# HS2SK-WiFi - Smart Metering Plug (WiFi)

## Overview

HS2SK-WiFi is a Wi-Fi smart metering plug launched by HEIMAN. It supports 2.4GHz 802.11 b/g/n direct connection to the home router, enabling remote control and energy monitoring without an additional gateway. Suitable for intelligent management of electrical appliances in homes, offices, and other scenarios.

## Technical Specifications

| Parameter | Value |
|------|-----|
| Communication Protocol | IEEE 802.11 b/g/n (2.4 GHz) |
| Maximum Load | 16A |
| Maximum Power | 3680W (230V) / 1800W (110V) |
| Metering Accuracy | ±2% |
| Power Supply | Mains powered |
| Standby Power | ≤ 0.5W |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤ 90% RH |
| Housing Material | PC V0 flame retardant |
| Socket Standard | EU / US (depending on market version) |

## Features

- Supports Smart Life / Tuya Smart App remote control
- Real-time power monitoring and energy statistics
- Remote on/off control
- Timer and countdown scheduling
- Scene linkage and automation
- Voice control (compatible with Alexa, Google Home)
