# Other Certifications

## Q-Label Quality Mark

Q-Label is a quality certification mark for the European smart home industry, launched by the Q-Lab Alliance (now part of the Zigbee Alliance / CSA Connectivity Standards Alliance).

| Item | Description |
|------|------|
| Scope of Application | Zigbee device interoperability certification |
| Certification Content | Compatibility of devices with other brands' Zigbee gateways/controllers |
| Significance | Ensures cross-brand interoperability, reducing the risk of "not working after purchase" |

Heiman Zigbee products are Q-Label certified, ensuring compatibility with the following platforms:
- Amazon Echo Plus / Echo Show (built-in Zigbee)
- Philips Hue Bridge
- SmartThings Hub
- Other Q-Label certified gateways
