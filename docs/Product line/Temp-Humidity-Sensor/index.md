# Temp/Humidity Sensor

The Temperature/Humidity Sensor monitors ambient temperature and relative humidity in real time, serving as a fundamental device for smart home environmental monitoring.

## Product Series

| Series | Communication | Features |
|------|----------|------|
| H1 Series | Matter / Zigbee | Matter over Thread version WWHA certified |

## Core Features

- Real-time temperature and humidity monitoring
- Low battery reminder
- Supports linkage control (turn on AC when hot, turn on dehumidifier when humid, etc.)

## Technical Specifications

- Temperature accuracy: ±0.3°C
- Humidity accuracy: ±3% RH
- Power supply: CR2 battery
- Battery life: ~2-3 years

## Scenario Examples

-linkage AC/heating to auto-adjust room temperature
-linkage dehumidifier/humidifier to maintain comfortable humidity
- Monitor wine cellar/cigar cabinet environment
- Kitchen/bathroom exhaust fan auto control
- Greenhouse environment monitoring
