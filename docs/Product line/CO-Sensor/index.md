# CO Sensor

Carbon monoxide (CO) is a colorless, odorless, non-irritating toxic gas known as the "silent killer." Heiman CO Sensor uses **electrochemical sensor technology** to accurately detect CO concentration in the air.

## Certification Achievements

Heiman launched the **world's first Matter-certified CO alarm**, which has also passed Works with Home Assistant certification.

## Core Features

- High-precision electrochemical CO sensor
- Detection range: 0~1000 ppm
- Tiered concentration alarm (audible and visual indicators)
- 85 dB alarm sound
- Low battery reminder
- Sensor end-of-life reminder (typically 7 years)
- Self-test function

## Safety Information

CO concentration and human response:

| CO Concentration (ppm) | Human Response |
|---------------|----------|
| 50 | Healthy adult can tolerate 8 hours |
| 200 | Mild headache after 2-3 hours |
| 400 | Headache and nausea after 1-2 hours |
| 800 | Dizziness and convulsions within 45 minutes |
| 1600+ | Fatal within 2 hours |
