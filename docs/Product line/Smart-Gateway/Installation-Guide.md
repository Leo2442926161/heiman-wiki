# Installation Guide

## Mounting Location Requirements

- **Central location in the home** to ensure Zigbee signal covers all devices
- Avoid placing inside all-metal cabinets (metal shields wireless signals)
- **Ethernet preferred**: wired connection recommended for better stability
- Within 20-30 m of other Zigbee devices (indoor)

## Installation Steps

1. Select mounting location (living room TV cabinet, near network cabinet, etc.)
2. Connect power adapter
3. Connect Ethernet cable (recommended) or connect to Wi-Fi
4. Wait for LED to turn green/solid
5. Open Heiman APP, scan QR code on gateway bottom to add
6. After adding, pair sub-devices

## Important Notes

- Gateway is the core of smart home, recommend using UPS backup power
- Ethernet connection is more stable than Wi-Fi, prioritize its use
- If device signal is weak, add Zigbee repeater devices (e.g. smart plug)
- Maximum sub-device count varies by model, choose according to scale
