# HS6GW - Smart Home Gateway

## Overview

HS6GW is a smart home gateway launched by HEIMAN, supporting Wi-Fi + Ethernet uplink, with a built-in Zigbee coordinator that can manage up to 128 Zigbee sub-devices. It enables remote access and cloud services through internet connectivity, serving as the core hub of the HEIMAN smart home system.

## Technical Specifications

| Parameter | Value |
|------|-----|
| Uplink Communication | IEEE 802.11 b/g/n (2.4 GHz Wi-Fi) + 10/100M Ethernet |
| Downlink Communication | Zigbee 3.0 (coordinator mode) |
| Maximum Sub-devices | 128 |
| Power Supply | DC 5V / 1A (Micro USB) |
| Standby Power | ≤ 1W |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤ 90% RH |
| Indicators | Wi-Fi status / Power / Zigbee network status |
| Reset Method | Button reset + long press factory reset |

## Features

- Dual uplink: Wi-Fi or Ethernet wired connection, flexible choice based on network environment
- Remote access: View device status anytime, anywhere via cloud App
- Local scene automation: Supports device automation within LAN (still functional offline)
- OTA firmware upgrade
- Supports Tuya / Smart Life platform
- Manages up to 128 Zigbee sub-devices
- Supports Zigbee network self-healing and route optimization
