# Installation Guide

## Mounting Location Requirements

- **Place on the floor**, ensure probes contact the ground
- **Near water sources**: under bathroom sink, under kitchen sink, next to washing machine, under water heater
- At the lowest point where water leakage needs to be detected

## Installation Steps

1. Determine installation location (floor area with leak risk)
2. Clean floor surface
3. Place sensor on floor, probes contacting the ground
4. If using cable version (HS2WL), lay extension cable along the area to be monitored
5. Insert battery, sensor powers on automatically
6. Simulation test: touch probes with wet finger, alarm LED lights up
7. Pair via gateway

## Important Notes

- Probes may corrode with prolonged contact with damp environments; inspect periodically
- Recommended to use with solenoid valve for automatic shutoff
- Battery life ~2-3 years
- Multiple sensors can cover different areas
