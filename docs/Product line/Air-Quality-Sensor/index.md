# Air Quality Sensor

The Air Quality Monitor detects multiple indoor air pollutants, helping maintain a healthy living environment.

## Detected Parameters

The Heiman air quality sensor can detect the following indicators:

| Parameter | Description | Health Standard |
|------|------|----------|
| PM2.5 | Fine particulate matter | ≤ 35 µg/m³ (24-hour average) |
| PM10 | Inhalable particles | ≤ 50 µg/m³ (24-hour average) |
| CO₂ | Carbon dioxide | ≤ 1000 ppm |
| TVOC | Total volatile organic compounds | ≤ 0.6 mg/m³ |
| Temperature | Ambient temperature | 18-26°C |
| Humidity | Relative humidity | 40-60% RH |

## Core Features

- Multi-parameter air quality detection
- Color display (model dependent)
- Out-of-range audible and visual alarm
- Historical data recording
-linkage fresh air system/air purifier

## Applicable Scenarios

- Bedroom: monitor CO₂ levels, prompt to open windows for ventilation
- Children's room: protect children's respiratory health
- Office: improve work efficiency and comfort
- Newly renovated home: monitor formaldehyde/TVOC emissions
