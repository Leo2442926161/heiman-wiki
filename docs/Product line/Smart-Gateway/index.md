# Smart Gateway

The Smart Gateway is the central hub of the Heiman smart home system, responsible for connecting various Zigbee sensors and enabling linkage control between devices.

## Product Series

| Series | Communication | Features |
|------|----------|------|
| HS1GW Series | Zigbee + Wi-Fi / Ethernet | Basic gateway |
| HS3GW Series | Zigbee + Wi-Fi / Ethernet | Enhanced |
| HS5GW Series | Zigbee + Wi-Fi / Ethernet | High-performance gateway |
| HS6GW Series | Zigbee + Wi-Fi / Ethernet + Thread | Latest model, supports both Zigbee and Matter over Thread |

## Core Features

- Zigbee network coordinator
- Wi-Fi / Ethernet internet connectivity
- Automatic device discovery and pairing
- Local scene linkage (select models)
- Remote access (via Heiman cloud platform)
- Supports up to 128 sub-devices (varies by model)

## Applicable Scenarios

- Core connection point for Heiman whole-home smart system
- Remote monitoring via Heiman APP
- Local centralized management for small to medium commercial projects

## Difference from Other Gateways

The Heiman gateway is the entry point to the official ecosystem. For users who prefer third-party platforms, it is recommended to use a universal Zigbee coordinator with Zigbee2MQTT / ZHA.
