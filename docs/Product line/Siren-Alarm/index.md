# Siren Alarm

The Smart Siren provides high-volume audible and visual alerts, working with various sensors as an output actuator for security systems.

## Core Features

- High-decibel alarm sound (≥110 dB)
- Red/color strobe light
- Multiple alarm tones supported
- Tamper detection
- Remote arm/disarm support

## Applicable Scenarios

-linkage with smoke/CO sensors: sounds throughout the house when danger is detected
- Security intrusion alarm: triggered when doors/windows are illegally opened
- Emergency evacuation: guides personnel evacuation during fire alarms

## Scene

- **Smoke alarm + siren**: simultaneous alarm throughout the house, alerting all room occupants
- **Door sensor alarm + siren**: deter illegal intruders
- **Gateway coordination**:linkage with other devices via Zigbee/Matter network
