## Contribution Guide
We welcome community contributions! Here's how to get involved.

### 📝 Contribute Documentation

#### Via GitHub Pull Request

1. Fork this repository
2. Modify or add documentation:
   - Documents are in the `docs/` and `blog/` directories
3. Submit a PR describing your changes
4. Wait for review and merge


#### Via GitHub Issues

Submit documentation suggestions on the [Issues page](https://github.com/Leo2442926161/heiman-wiki/issues):

- Click the **"Doc Suggestion"** template
- Fill in the suggestion content and related page paths
- We will evaluate and process it after submission

### 🐛 Report Bugs

If you encounter issues, please submit a GitHub Issue:

1. Go to the [Issues page](https://github.com/Leo2442926161/heiman-wiki/issues/new/choose)
2. Select the **"Bug Report"** template
3. Fill in:
   - The page path where the error occurred
   - Expected behavior vs. actual behavior
   - Screenshots (optional)
   - Browser and language version
4. Submit the Issue

### 💬 Article Comments

This site provides commenting functionality based on GitHub Discussions. Visit the [Discussions page](https://github.com/Leo2442926161/heiman-wiki/discussions) to join the conversation.


### Translation Improvements

Found inaccurate or improvable translations? Feel free to:

- Submit a PR directly to modify files under `docs/`
- Select the "Translation Improvement" type in an Issue to submit feedback
