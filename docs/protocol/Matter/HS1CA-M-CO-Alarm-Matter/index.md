# HS1CA-M Smart Carbon Monoxide Alarm (Matter)

HS1CA-M is the world's **first Matter-certified carbon monoxide alarm**, utilizing electrochemical sensor technology and supporting Matter over Thread cross-ecosystem interoperability.

## Product Overview

- **Product Type**: Smart Carbon Monoxide Alarm
- **Communication Protocol**: Matter over Thread
- **Certification Standard**: EN 50291
- **Typical Scenarios**: CO leak early warning in homes, apartments, etc.

## Technical Parameters

| Parameter | Value |
|------|-----|
| Sensor Type | Electrochemical CO Sensor |
| Communication Protocol | Matter over Thread (IEEE 802.15.4) |
| Operating Frequency | 2.4 GHz |
| Detection Range | 0~1000 ppm |
| Alarm Sound Pressure | ≥85 dB (3 m) |
| Power Supply | DC 3V (CR123A Lithium Battery) |
| Battery Life | Approx. 3 years |
| Sensor Life | Typical 7 years |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH (non-condensing) |

## Features

- High-precision electrochemical CO sensor
- Concentration-graded alarm (audible and visual indication)
- 85 dB loud audible and visual alarm
- Low battery reminder
- Sensor end-of-life reminder
- Self-test function
- Matter multi-ecosystem cross-platform interoperability

## Certifications

- EN 50291 Certified
- Matter Certified
