# Supported Models List

HEIMAN Wi-Fi devices are primarily designed for direct connection scenarios, requiring no dedicated gateway to access the home network.

## Smart Plugs

| Model | Product Name | Description |
|------|----------|------|
| HS2SK-WiFi | Smart Plug (WiFi) | Direct connection to 2.4GHz Wi-Fi, supports remote control and scheduling |

## Security Alarms

| Model | Product Name | Description |
|------|----------|------|
| HS8SA-WiFi | Smart Smoke Alarm (WiFi US Version) | US market Wi-Fi version smoke alarm |

## Gateways

| Model | Product Name | Description |
|------|----------|------|
| HS6GW | Smart Home Gateway (WiFi + Zigbee) | Connects to internet via Wi-Fi, manages Zigbee sub-devices |
| HS6GW-2 | Smart Home Gateway (WiFi + Zigbee) | Second-generation gateway, upgraded performance |

## Notes

- All Wi-Fi devices only support the **2.4 GHz** band, not compatible with 5 GHz
- It is recommended to use WPA2-PSK encrypted networks
- Internet connection requires the router to be properly connected to the internet
