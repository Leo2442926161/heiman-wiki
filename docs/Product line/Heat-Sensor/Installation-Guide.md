# Installation Guide

## Mounting Location Requirements

- **Ceiling mount**, center of room
- **Best use scenarios**: kitchens, garages, boiler rooms, bathrooms — where smoke sensors are prone to false alarms
- At least **30 cm** from walls
- Avoid installing near vents, AC outlets

## Installation Steps

1. Select mounting location (recommended: kitchen or garage ceiling)
2. Secure mounting base to ceiling
3. Insert battery, sensor powers on automatically
4. Rotate and snap main unit into base
5. Self-test: press test button, LED lights up and sounds
6. Pair via gateway (Zigbee version)

## Important Notes

- Heat sensor is **not a replacement** for smoke sensors, but a complement
- Smoke sensors respond faster, heat sensors have better interference immunity
- Recommend installing both smoke and heat sensors in the same room
- Test once a month, replace battery promptly when low
