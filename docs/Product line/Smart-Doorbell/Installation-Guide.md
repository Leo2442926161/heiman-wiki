# Installation Guide

## Mounting Location Requirements

- Install at doorbell height approximately **1.4 ~ 1.6 m** (at eye level)
- Face the doorway passage to ensure visitor's face is captured
- Avoid direct sunlight on the camera

## Installation Steps

1. Select mounting location (next to main door or on door frame)
2. Secure mounting base with screws
3. Connect power (wired version: connect wires; battery version: insert battery)
4. Install doorbell body onto base
5. Pair via APP and connect to Wi-Fi (Wi-Fi version)
6. Adjust camera angle
7. Test doorbell button, video, and audio functions

## Important Notes

- Wi-Fi version requires good Wi-Fi signal coverage at the door
- For battery version,note charging cycle; recommend using with door sensor (wake on door open)
- Clean camera lens periodically
- Ensure doorbell is within effective Zigbee signal range
