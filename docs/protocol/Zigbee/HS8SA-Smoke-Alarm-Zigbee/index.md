# HS8SA - Smart Smoke Alarm (Zigbee)

## Overview

The HS8SA is Heiman's Zigbee + Wi-Fi smart smoke alarm designed for the US market, featuring high-temperature linkage functionality and UL 217 certification.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 / Wi-Fi |
| Sensor Type | Infrared Photoelectric |
| Alarm Sound Level | ≥85 dB (3 m) |
| Power Supply | DC 3V (CR123A) |
| Operating Temperature | -10°C ~ +50°C |
| Certification Standard | UL 217 |
| Target Market | USA |

## Features

- Photoelectric smoke detection
- High-temperature linked alarm
- 85 dB audible and visual alarm
- Low battery reminder
- Self-test function
- Tamper alarm
- Remote silence support
- Dual protocol communication (Zigbee + Wi-Fi)

## Certification Standards

- UL 217 (US smoke alarm standard)
