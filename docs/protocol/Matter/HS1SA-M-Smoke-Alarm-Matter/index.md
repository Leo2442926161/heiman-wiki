# HS1SA-M Smart Smoke Alarm (Matter)

HS1SA-M is a **Matter over Thread** smart smoke alarm launched by Heiman, using photoelectric smoke detection technology and supporting cross-ecosystem smart home platform interoperability.

## Product Overview

- **Product Type**: Smart Smoke Alarm
- **Communication Protocol**: Matter over Thread
- **Certification Standard**: EN 14604
- **Typical Scenarios**: Fire early warning in homes, apartments, hotels, etc.

## Technical Parameters

| Parameter | Value |
|------|-----|
| Sensor Type | Infrared Photoelectric |
| Communication Protocol | Matter over Thread (IEEE 802.15.4) |
| Operating Frequency | 2.4 GHz |
| Alarm Sound Pressure | ≥85 dB (3 m) |
| Power Supply | DC 3V (CR123A Lithium Battery) |
| Battery Life | Approx. 3 years |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH (non-condensing) |

## Features

- Photoelectric smoke detection, capable of detecting smoldering and open flame smoke
- 85 dB loud audible and visual alarm
- Low battery reminder
- Self-test function (one-button test)
- Tamper alarm
- Remote silence
- Matter multi-ecosystem cross-platform interoperability

## Certifications

- EN 14604 Certified
- Matter Certified
