# HS1SA-E - Smart Smoke Alarm (Zigbee)

## Overview

The HS1SA-E is Heiman's standard Zigbee smart smoke alarm designed for the EU market. It uses photoelectric smoke detection technology, is EN 14604 certified, and has won the Red Dot Design Award.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Infrared Photoelectric |
| Alarm Sound Level | ≥85 dB (3 m) |
| Power Supply | DC 3V (CR123A) |
| Battery Life | Approx. 3 years |
| Communication Range | ≤70 m (open area) |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH |
| Installation Method | Ceiling Mount |

## Features

- Photoelectric smoke detection, capable of detecting smoke from smoldering and open flames
- 85 dB audible and visual alarm
- Low battery reminder
- Self-test function (one-button test)
- Tamper alarm
- Remote silence support
- OTA firmware upgrade

## Certification Standards

- EN 14604 (European Standard for Smoke Alarms)
- Red Dot Design Award
- CE / RED

## Compatible Accessories

- CR123A battery
- Mounting screws / wall plugs
