# HS6GW - Smart Gateway (Zigbee)

## Overview

The HS6GW is Heiman's Zigbee smart gateway, serving as the coordinator of the Heiman Zigbee network. It connects to the internet via Wi-Fi or Ethernet uplink and supports up to 128 sub-devices.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 (Coordinator) |
| Uplink Connection | Wi-Fi / Ethernet |
| Max Nodes | 128 sub-devices |
| Power Supply | DC 5V USB Powered |

## Features

- Zigbee network coordinator
- Wi-Fi / Ethernet uplink
- Automatic device discovery and pairing
- Local scene automation
- Remote access (via Heiman cloud platform)
- Supports up to 128 sub-devices
- OTA firmware upgrade

## Application Scenarios

- Central hub for Heiman whole-home smart system
- Works with Heiman APP for remote monitoring
