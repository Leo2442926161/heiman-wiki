# Installation Guide

## Mounting Location Requirements

- Wall mount height: **1.8 ~ 2.4 m**
- Ceiling mount: install at center of room ceiling
- **Avoid**: direct sunlight, heater/AC vents, near heat sources
- Avoid obstructions within detection range (furniture, glass partitions)

## Installation Steps

1. Select mounting location (wall or ceiling)
2. Secure mounting base with screws
3. Insert battery, LED flashes
4. Install sensor body onto base
5. Adjust detection angle (within 110° horizontal range)
6. Walk test: walk through detection area, LED should illuminate
7. Pair via gateway

## Important Notes

- PIR sensors are sensitive to temperature changes; detection distance may decrease in hot summer environments
- Avoid facing windows directly (outdoor temperature changes may cause false alarms)
- Battery life ~3 years (Zigbee) / 1 year (Matter)
