# HS1WL - Water Leak Detector (Zigbee)

## Overview

The HS1WL is Heiman's first generation Zigbee water leak detector, using probe-based detection to effectively detect water leaks or accumulation. It is a basic water monitoring device.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Detection Method | Probe Detection |
| Power Supply | DC 3V Battery |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH |

## Features

- Real-time water leak / accumulation detection
- Buzzer alarm
- Low battery reminder
- Probe anti-corrosion design
