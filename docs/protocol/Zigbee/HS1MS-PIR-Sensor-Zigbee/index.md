# HS1MS - PIR Motion Sensor (Zigbee)

## Overview

The HS1MS is Heiman's standard Zigbee PIR (Passive Infrared) motion sensor, using passive infrared detection technology to sense infrared heat radiation emitted by moving people. It is widely used in smart lighting and security scenarios.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Passive Infrared (PIR) |
| Detection Angle | 110° |
| Detection Distance | ≤7 m (25°C) |
| Power Supply | DC 3V Lithium Battery |
| Battery Life | Approx. 3 years |
| Operating Temperature | -10°C ~ +50°C |
| Installation Method | Wall Mount |

## Features

- Motion detection
- 110° wide-angle detection
- Low battery reminder
- Tamper alarm
- Supports scene automation
