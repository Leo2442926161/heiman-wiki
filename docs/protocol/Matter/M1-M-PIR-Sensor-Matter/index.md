# M1-M PIR Sensor (Matter)

M1-M is a **Matter over Thread** PIR sensor launched by Heiman, integrating PIR detection and ambient light sensing, suitable for smart home automation scenarios.

## Product Overview

- **Product Type**: PIR Sensor
- **Communication Protocol**: Matter over Thread
- **Detection Method**: Passive Infrared (PIR) + Ambient Light Detection
- **Typical Scenarios**: Light-on-when-occupant-detected, security integration, environmental automation

## Technical Parameters

| Parameter | Value |
|------|-----|
| Detection Angle | 110° (Wall-mounted) / 360° (Ceiling-mounted) |
| Detection Distance | ≤7 m (25°C) |
| Ambient Light Range | 0~1200 Lux |
| Communication Protocol | Matter over Thread (IEEE 802.15.4) |
| Operating Frequency | 2.4 GHz |
| Power Supply | DC 3V Lithium Battery |
| Battery Life | Approx. 1 year (Matter mode) |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH (non-condensing) |

## Features

- Motion detection
- Ambient light detection
- Detection distance ≤7m
- Supports wall or ceiling mounting
- Matter multi-ecosystem cross-platform interoperability
