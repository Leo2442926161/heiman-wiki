# Heat Sensor

The Heat Detector detects abnormal temperature rises to trigger fire alarms, suitable for kitchens, garages, and other scenarios where smoke sensors are prone to false alarms.

## Working Principle

The heat sensor uses thermosensitive elements to detect temperature changes:

- **Fixed temperature alarm**: triggers when temperature exceeds set threshold (typically 57°C / 135°F)
- **Rate-of-rise alarm**: triggers when temperature rise rate exceeds set value (e.g. 8.3°C/minute)

## Applicable Scenarios

- Kitchens (smoke sensors are prone to false alarms from cooking)
- Garages/workshops (exhaust fumes, dust may interfere with smoke sensors)
- Boiler rooms/heating equipment rooms
- Bathrooms/saunas

## Core Features

- Abnormal temperature rise detection
- Tamper alarm
- Low battery reminder

> **Note**: Heat sensors and smoke sensors are complementary. Smoke sensors respond faster, while heat sensors have better anti-interference capability.
