# Homey

Homey is a smart home hub developed by the Dutch company Athom, supporting Zigbee, Z-Wave, Wi-Fi, Matter, Thread and other protocols.

## Integration

Heiman Zigbee and Z-Wave devices can be paired directly via Homey's built-in wireless chip, no additional gateway required. Matter devices can be connected through Homey Pro's built-in Thread border router.

## Advantages

- **Multi-protocol all-in-one**: Homey Pro supports Zigbee, Z-Wave, Thread, and Wi-Fi simultaneously
- **Local operation**: All automations run locally
- **Flow Automation**: Drag-and-drop programming to create complex automation scenarios

> **Reference**: Search for Heiman-related apps in the Homey App Store
