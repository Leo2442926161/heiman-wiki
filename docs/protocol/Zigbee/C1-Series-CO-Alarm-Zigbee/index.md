# C1 Series - Carbon Monoxide Alarm (Zigbee)

## Overview

The C1 series is Heiman's entry-level Zigbee carbon monoxide alarm, featuring an electrochemical sensor for CO leak detection scenarios, certified to EN 50291.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Electrochemical |
| Detection Range | 0 ~ 1000 ppm |
| Alarm Sound Level | ≥85 dB (3 m) |
| Power Supply | Battery |
| Sensor Life | Approx. 7 years |
| Operating Temperature | -10°C ~ +50°C |

## Features

- Electrochemical CO detection
- Concentration-graded alarm
- 85 dB audible and visual alarm
- Low battery reminder
- Self-test function

## Certification Standards

- EN 50291
