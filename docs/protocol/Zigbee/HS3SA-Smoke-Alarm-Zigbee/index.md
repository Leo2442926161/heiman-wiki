# HS3SA - Smart Smoke Alarm (Zigbee)

## Overview

The HS3SA is Heiman's 3rd generation Zigbee smart smoke alarm, featuring enhanced detection performance and linkage capabilities over the first generation, designed for the EU market.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Infrared Photoelectric |
| Alarm Sound Level | ≥85 dB (3 m) |
| Power Supply | DC 3V (CR123A) |
| Battery Life | Approx. 3 years |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH |
| Installation Method | Ceiling Mount |

## Features

- Photoelectric smoke detection
- 85 dB audible and visual alarm
- Low battery reminder
- Self-test function
- Tamper alarm
- Remote silence support
- OTA firmware upgrade
- Enhanced wireless performance

## Certification Standards

- EN 14604
- CE / RED

## Compatible Accessories

- CR123A battery
