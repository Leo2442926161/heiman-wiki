# Vibration Sensor

The Vibration Sensor detects vibration or tilt of objects to trigger alarms, suitable for doors, windows, safes, pipes, and various other scenarios.

## Core Features

- 3-axis vibration detection
- Tilt angle detection
- Low battery reminder

## Applicable Scenarios

- **Door/window tamper detection**: detects vibration from forced entry
- **Safe/drawer monitoring**: detects abnormal movement
- **Pipe monitoring**: detects pipe vibration (indicating pump abnormality or water hammer)
- **Wall/fence monitoring**: detects climbing or impact

## Technical Specifications

- Sensor type: 3-axis accelerometer
- Detection dimensions: vibration + tilt
- Power supply: CR2450 coin cell battery
- Battery life: ~2 years
