# Model Comparison

## Model Comparison

| Model | Protocol | Sensor | Power | Features | Usage Suggestions |
|------|------|--------|------|------|----------|
| Vibration Sensor | Zigbee | 3-axis accelerometer | CR2450 | Vibration + tilt detection | Door/window tamper, safe/drawer illegal open vibration alarm, pipe abnormal vibration monitoring |

## Scenario Recommendations

| Scenario | Recommended Model | Automation Scheme |
|------|----------|----------|
| Door/window tamper | Vibration Sensor | Detect forced door/window vibration → push alert + siren + camera snapshot |
| Safe monitoring | Vibration Sensor | Safe moved or vibrated → immediate alarm notification |
| Pipe monitoring | Vibration Sensor | Monitor abnormal pipe vibration (pump failure/water hammer), early warning maintenance |
| Fence/wall | Vibration Sensor | Wall/fence climbing detection, suitable for yard security |
