# HS1CG - Combustible Gas Detector (Zigbee)

## Overview

The HS1CG is Heiman's Zigbee combustible gas detector, featuring a semiconductor sensor that can detect natural gas (methane) and liquefied petroleum gas (LPG) leaks. It is certified to GB 15322 and supports linked solenoid valve automatic shut-off.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Semiconductor |
| Detected Gases | Natural Gas (Methane) / Liquefied Petroleum Gas (LPG) |
| Alarm Sound Level | ≥85 dB (3 m) |
| Power Supply | AC 220V / DC 12V |
| Certification Standard | GB 15322 |
| Operating Temperature | -10°C ~ +50°C |

## Features

- Natural gas / LPG leak detection
- Concentration-graded alarm
- Automatic shut-off linkage (with solenoid valve)
- 85 dB audible and visual alarm
- Self-test function
- Sensor end-of-life reminder

## Certification Standards

- GB 15322 (Chinese standard for combustible gas detectors)

## Compatible Accessories

- JT series gas emergency shut-off valve
