# Presence Sensor

The Presence Sensor (Occupancy / Presence Sensor) uses **millimeter-wave radar technology** to detect stationary human presence, complementing traditional PIR infrared sensors.

## Product Series

| Series | Communication | Features |
|------|----------|------|
| HS8OS | Zigbee | Millimeter-wave radar presence detection |

## Technology Principle

Unlike PIR sensors that can only detect movement, presence sensors transmit and receive millimeter-wave signals, using the micro-Doppler effect to detect tiny body movements (such as chest movements from breathing), thereby determining if someone is present.

## Core Features

- **Stationary presence detection**: detects lying down, sitting still, and other micro-movements
- **Detection range**: covers radius of 3-5 meters
- **Ceiling mount**: optimal detection angle
- **Complements PIR**: PIR handles wide-area motion detection, presence sensor handles fine-grained detection

## Scenario Examples

- **Smart lighting**: lights stay on when someone is present, auto-off when vacant (solves PIR lights-off issue when stationary)
- **HVAC control**: auto on/off based on room occupancy
- **Elderly care**: prolonged stillness in bathroom/bedroom triggers system alert
- **Office energy saving**: auto-adjust energy when meeting rooms/workstations are unoccupied
