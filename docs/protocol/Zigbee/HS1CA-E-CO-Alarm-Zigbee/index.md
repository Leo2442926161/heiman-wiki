# HS1CA-E - Smart Carbon Monoxide Alarm (Zigbee)

## Overview

The HS1CA-E is Heiman's Zigbee smart carbon monoxide (CO) alarm for the EU market, featuring an electrochemical sensor that can accurately detect CO concentrations from 0 to 1000 ppm, certified to EN 50291.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Electrochemical |
| Detection Range | 0 ~ 1000 ppm |
| Alarm Sound Level | ≥85 dB (3 m) |
| Power Supply | DC 3V (CR123A) |
| Sensor Life | Approx. 7 years |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH |
| Installation Method | Wall Mount |

## Features

- High-precision electrochemical CO sensor
- Concentration-graded alarm (audible and visual indicators)
- 85 dB alarm sound
- Low battery reminder
- Sensor end-of-life reminder
- Self-test function
- OTA firmware upgrade

## Certification Standards

- EN 50291 (European standard for CO alarms)
- CE / RED

## Compatible Accessories

- CR123A battery
