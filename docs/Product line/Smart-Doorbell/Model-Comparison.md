# Model Comparison

## Model Comparison

| Model | Protocol | Video | Power | Features | Usage Suggestions |
|------|------|------|------|------|----------|
| Smart Doorbell | Zigbee + Wi-Fi | 1080P optional | Battery/Mains | Doorbell + security 2-in-1 | Visitor ring notifies phone, remote video intercom, motion detection recording |

## Scenario Recommendations

| Scenario | Recommended Model | Automation Scheme |
|------|----------|----------|
| Delivery/takeout management | Smart Doorbell | Ring → phone notification → remote intercom confirm → remote unlock (linkage smart lock) |
| Security monitoring | Smart Doorbell | Front door loitering detection → recording + push alert,linkage door sensor/lighting |
| Visitor linkage | Smart Doorbell | Ring → indoor screen/speaker shows video → turn on light → unlock linkage |
