# Model Comparison

## Model Comparison

| Model | Protocol | Detection Parameters | Power | Features | Usage Suggestions |
|------|------|----------|------|------|----------|
| Air Quality Sensor | Zigbee | PM2.5 / PM10 / CO₂ / TVOC / Temp/Humidity | Mains | Multi-function all-in-one | Monitor indoor air quality, auto linkage fresh air/purifier + push alert when exceeds thresholds |

## Scenario Recommendations

| Scenario | Recommended Model | Automation Scheme |
|------|----------|----------|
| Bedroom CO₂ monitoring | Air Quality Sensor | CO₂ > 1000ppm → window reminder /linkage fresh air system for ventilation, better sleep quality |
| Newly renovated home | Air Quality Sensor | Continuous TVOC/formaldehyde monitoring,linkage exhaust fan when exceeds thresholds, push reassurance notice when meets standards |
| Children's room | Air Quality Sensor | PM2.5 monitoring, auto turn on air purifier when exceeds thresholds, protect respiratory health |
| Office | Air Quality Sensor | CO₂ + temp/humidity linkage AC fresh air, improve work efficiency |
