# Zigbee

Zigbee is a low-power, low-data-rate wireless Mesh network protocol based on the IEEE 802.15.4 standard. It is one of the most mature and widely adopted protocols in the smart home domain.

## Heiman and Zigbee

Heiman is among the first Chinese manufacturers to adopt the Zigbee protocol, with an extensive Zigbee product line. Heiman Zigbee devices are compatible with the **Zigbee 3.0** standard and can interoperate with mainstream Zigbee gateways and coordinators.

## Technical Specifications

| Feature | Description |
|---------|-------------|
| Frequency Band | 2.4 GHz (Global) |
| Data Rate | Up to 250 kbps |
| Network Topology | Mesh Self-Healing Network |
| Node Capacity | Up to 65,535 nodes per network |
| Communication Range | Approx. 10-30 m indoors (depending on obstacles) |
| Power Consumption | Ultra-low; sensor battery life up to 3 years |

## Compatible Platforms

Heiman Zigbee devices can be integrated with the following platforms:

- **Zigbee2MQTT** — Open-source Zigbee to MQTT bridge
- **ZHA** — Home Assistant built-in Zigbee integration
- **Hubitat** — Local smart home hub
- **Homey** — Multi-protocol smart home hub
- **SmartThings Hub** — Samsung smart home hub
- **Tuya Zigbee Gateway** — Tuya ecosystem
- **Heiman HS6GW Gateway** — Official gateway
