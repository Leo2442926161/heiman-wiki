# Certification Standards

HS8SA has obtained the following certifications:

- **EN 14604** — European standard for smoke alarms (EU model), see [EN European Standards](../../../../standard/EN-European-Standards.md)
- **UL 217** — US smoke alarm safety standard (US model), see [UL Underwriters Laboratories](../../../../standard/UL-Underwriters-Laboratories.md)
- **CE / RED / EMC** — EU radio equipment directive and electromagnetic compatibility
- **Zigbee 3.0 / Q-Label** — Zigbee Alliance certification, see [Other Certifications](../../../standard/Other-Certifications.md)
