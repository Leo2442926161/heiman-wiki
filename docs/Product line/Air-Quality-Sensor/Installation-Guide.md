# Installation Guide

## Mounting Location Requirements

- Install at **1.2 ~ 1.8 m** height (breathing zone)
- Choose a location representative of room average air quality
- **Keep away from**: windows, doors, vents, AC outlets
- **Keep away from**: kitchen fumes, humidifiers, aromatherapy, and other local pollution sources

## Installation Steps

1. Select mounting location (living room, bedroom, or office wall)
2. Secure mounting base with screws
3. Connect power (USB or adapter powered)
4. Sensor powers on, display (if available) shows data after self-test
5. Pair via gateway
6. Confirm data is normal in APP

## Important Notes

- Ensure sensor air intake and exhaust are not blocked
- Clean sensor housing periodically to prevent dust buildup
- Place in children's room to monitor CO₂ levels and prompt window ventilation
- Recommended to linkage with fresh air system or air purifier
