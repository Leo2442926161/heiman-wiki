# Door/Window Sensor

The Door/Window Sensor detects the open/close status of doors and windows, serving as a fundamental component of smart security systems.

## Product Series

| Series | Communication | Features |
|------|----------|------|
| HS1DS Series | Zigbee / Z-Wave | Standard type, 3-year battery life |
| HS1DS-M | Matter over Thread | Matter version |

## Core Features

- Real-time door/window open/close status detection
- Tamper alarm
- Low battery reminder
- Supports linkage (open door turn on light, away arming, etc.)
- Easy installation (3M adhesive or screws)

## Technical Specifications

- Power supply: CR1632 coin cell battery
- Battery life: ~2-3 years
- Operating temperature: -10°C ~ +50°C
- Mounting gap: ≤10 mm

## Scenario Examples

- **Away arming**: door sensor triggers alarm on opening
- **Arrival scene**: door opens, lights turn on automatically
- **Window monitoring**:linkage weather station, close window reminder when it rains
