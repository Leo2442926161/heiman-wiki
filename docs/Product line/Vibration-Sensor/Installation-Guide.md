# Installation Guide

## Mounting Location Requirements

- Mount on the surface of the object to be monitored for vibration
- Common mounting locations: **inside of doors/windows**, **safes**, **pipes**, **walls**, **fences**
- Ensure mounting surface is flat and clean

## Installation Steps

1. Select object to monitor (e.g. inside of safe door)
2. Clean mounting surface
3. Secure sensor with 3M adhesive or screws
4. Insert CR2450 battery
5. Pair via gateway
6. Simulation test: tap mounting surface, sensor should trigger alarm

## Important Notes

- For door/window tamper scenarios, install on the inside near the lock for best results
- For pipe monitoring, attach directly to pipe surface
- Battery life ~2 years
- Sensitivity adjustable for different scenarios
