# HS8MLS - Micro Motion Sensor (Zigbee)

## Overview

The HS8MLS is Heiman's long-range Zigbee PIR motion sensor, designed for scenarios requiring long-distance detection, suitable for corridors, halls, and other open spaces.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Passive Infrared (PIR) |
| Detection Distance | Long range detection |
| Power Supply | DC 3V Battery |

## Features

- Long-range motion detection
- Low power design
- Supports scene automation
