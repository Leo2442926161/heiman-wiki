# Model Comparison

## Model Comparison

| Model | Protocol | Alarm Sound Pressure | Power | Features | Usage Suggestions |
|------|------|----------|------|------|----------|
| Siren Alarm | Zigbee | ≥110 dB | Mains | Bright strobe + high volume | Smoke/CO/door sensor triggers whole-house siren, deters intruders when away-armed |

## Scenario Recommendations

| Scenario | Recommended Model | Automation Scheme |
|------|----------|----------|
| Whole-house fire linkage | Siren Alarm | Any smoke/CO sensor alarm → all sirens sound throughout house, waking sleeping occupants |
| Security intrusion deterrence | Siren Alarm | Door sensor illegal open + away mode → siren sounds + all lights on, deterring intruders |
| Emergency evacuation guidance | Siren Alarm | Fire alarm triggered → siren guides evacuation direction, use with door sensor to check if evacuated |
