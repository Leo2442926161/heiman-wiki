# HS1DS - Door/Window Sensor (Zigbee)

## Overview

The HS1DS is Heiman's Zigbee door/window sensor, used to detect the open/close status of doors and windows, serving as a fundamental component of smart security systems.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Reed Switch |
| Power Supply | CR1632 coin cell battery |
| Battery Life | Approx. 2 ~ 3 years |
| Installation Gap | ≤10 mm |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH |

## Features

- Real-time door/window open/close status detection
- Tamper alarm
- Low battery reminder
- Supports automation (open door turn on light, away mode arm, etc.)
- Easy installation (3M adhesive or screw mount)

## Compatible Accessories

- CR1632 coin cell battery
- 3M double-sided tape / screws
