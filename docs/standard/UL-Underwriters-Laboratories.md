# UL Underwriters Laboratories

UL (Underwriters Laboratories) is the most authoritative product safety standard development and certification organization in the United States. The main UL standards relevant to Heiman products are as follows:

## UL 217 — Smoke Alarm Standard (USA)

| Item | Description |
|------|------|
| Scope of Application | Residential smoke alarm (US market) |
| Test Requirements | Photoelectric/ionization smoke detection, audible and visual alarm, 85 dB or above |
| Applicable Markets | USA, Canada |
| Heiman Products | HS8SA (US version) |
| Certification Significance | Meets US state fire safety regulations, a prerequisite for North American market access |

**Difference from other standards**: UL 217 test scenarios include flash fire, smoldering fire and other typical US household fire scenarios, differing slightly from EN 14604 test methods.

## UL 2034 — Carbon Monoxide Alarm Standard (USA)

| Item | Description |
|------|------|
| Scope of Application | Household CO alarm (US market) |
| Test Requirements | Electrochemical sensor, concentration × time integral algorithm |
| Applicable Markets | USA, Canada |
| Heiman Products | HS-720ES-M (US version) |
| Certification Significance | Most US states mandate CO alarm installation |

## Key Differences between UL and EN Certification

| Aspect | UL | EN |
|------|-----|-----|
| Smoke alarm standard | UL 217 | EN 14604 |
| CO alarm standard | UL 2034 | EN 50291 |
| Test method | Specific fire curve test | Different fire type test |
| Applicable voltage | 120V / Battery | 230V / Battery |
