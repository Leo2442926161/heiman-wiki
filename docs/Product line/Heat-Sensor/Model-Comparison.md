# Model Comparison

## Model Comparison

| Model | Protocol | Detection Principle | Power | Features | Usage Suggestions |
|------|------|----------|------|------|----------|
| Heat Sensor | Zigbee / Sub-1G | Thermistor (fixed temp + rate-of-rise) | Battery | Anti-smoke interference | Kitchen/garage-dedicated, abnormal temp rise alarm,linkage solenoid valve to shut off gas |

## Scenario Recommendations

| Scenario | Recommended Model | Automation Scheme |
|------|----------|----------|
| Kitchen safety | Heat Sensor | Abnormal temp rise → close gas valve → siren alarm → phone push, avoid cooking smoke false alarm |
| Garage/boiler room | Heat Sensor | Monitor equipment room temperature, prevent spontaneous combustion,linkage exhaust fan to cool |
| Complement smoke sensor | Heat + Smoke | Smoke sensor covers whole house, heat sensor covers kitchen/garage, complementary alarm for reliability |

## Selection Guide

- **Kitchen-dedicated**: Heat Sensor (avoid cooking smoke false alarms)
- **Use with smoke sensor**: Smoke sensor covers bedrooms/living room, heat sensor covers kitchen/garage
