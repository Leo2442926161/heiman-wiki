# Matter

Matter (formerly Project CHIP) is a smart home interoperability standard launched by the Connectivity Standards Alliance (CSA), designed to break down brand barriers and enable cross-ecosystem interoperability.

## Heiman and Matter

Heiman is an early adopter of the Matter protocol, launching the **world's first Matter-certified carbon monoxide alarm**. Heiman's Matter devices all use the **Matter over Thread** solution.

## Technical Features

| Feature | Description |
|------|------|
| Underlying Protocol | Thread (Low-power Mesh Network) |
| Network Topology | Mesh self-healing network, wider coverage with more devices |
| Power Consumption | Ultra-low, battery-powered operation for 1-3 years |
| Security | Supports AES-128-CCM encryption, device certification required |
| Interoperability | Across Apple HomeKit, Google Home, Amazon Alexa, Home Assistant, etc. |

## Border Router Requirements

Using Matter over Thread devices requires a Thread Border Router:

- Apple HomePod mini / HomePod 2nd Gen / Apple TV 4K
- Google Nest Hub 2nd Gen / Nest Hub Max
- Amazon Echo 4th Gen
- Home Assistant Green + Connect ZBT-2
- Homey Pro
