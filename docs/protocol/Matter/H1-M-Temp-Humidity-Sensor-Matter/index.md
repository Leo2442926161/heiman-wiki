# H1-M Temperature and Humidity Sensor (Matter)

H1-M is a **Matter over Thread** temperature and humidity sensor launched by Heiman, capable of real-time monitoring of ambient temperature and relative humidity, supporting smart home automation triggers.

## Product Overview

- **Product Type**: Temperature and Humidity Sensor
- **Communication Protocol**: Matter over Thread
- **Detection Parameters**: Temperature, Relative Humidity
- **Typical Scenarios**: Environmental monitoring in homes, offices, wine cellars, etc.

## Technical Parameters

| Parameter | Value |
|------|-----|
| Temperature Accuracy | ±0.3°C |
| Humidity Accuracy | ±3% RH |
| Communication Protocol | Matter over Thread (IEEE 802.15.4) |
| Operating Frequency | 2.4 GHz |
| Power Supply | CR2 Lithium Battery |
| Battery Life | Approx. 2~3 years |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH (non-condensing) |

## Features

- Real-time temperature and humidity monitoring
- High-precision sensing (Temperature ±0.3°C, Humidity ±3% RH)
- Low battery reminder
- Matter multi-ecosystem cross-platform interoperability
