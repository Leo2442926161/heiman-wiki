# HS8SA-WiFi - Smart Smoke Alarm (WiFi US Version)

## Overview

HS8SA-WiFi is a Wi-Fi smart smoke alarm launched by HEIMAN for the US market. It uses photoelectric smoke detection technology, supports both Wi-Fi联网 and standalone operation modes, and is UL 217 certified. Suitable for fire safety protection in homes, apartments, and other residential spaces.

## Technical Specifications

| Parameter | Value |
|------|-----|
| Communication Protocol | IEEE 802.11 b/g/n (2.4 GHz) |
| Sensor Type | Infrared photoelectric |
| Alarm Sound Level | ≥85 dB (3 meters) |
| Power Supply | Battery powered |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤ 90% RH |
| Installation Method | Ceiling mount |

## Features

- Photoelectric smoke detection, capable of detecting smoke from smoldering and open flames
- Wi-Fi connected remote alarm push notifications
- Standalone operation mode (local audible and visual alarm still functions when offline)
- 85 dB loud audible and visual alarm
- Low battery notification
- Self-test function (one-button test)
- Tamper alarm
- Remote silence support

## Certification Standards

- UL 217 (US Standard for Smoke Alarms)
- FCC
- NFPA 72 compliant
