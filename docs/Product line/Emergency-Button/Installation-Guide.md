# Installation Guide

## Mounting Location Requirements

- **Wall mount**: bathroom, bedroom bedside, next to living room sofa — within easy reach
- **Wearable**: can be placed in pocket or attached to keychain
- **For elderly**: install in bathroom and bedside, at approximately 1.2 m height

## Installation Steps

1. Determine mounting location or whether to carry
2. Wall mount: secure base with screws or 3M adhesive
3. Insert battery
4. Place emergency button on base or carry with you
5. Pair via gateway
6. Test: long press button to trigger alarm, confirm APP receives notification

## Important Notes

- Uses **long press anti-accidental-touch** design, requires continuous press for 1-2 seconds to trigger
- Recommended to linkage with siren alarm
- Test functionality periodically
- For elderly use, can pair with fall detector for dual protection
