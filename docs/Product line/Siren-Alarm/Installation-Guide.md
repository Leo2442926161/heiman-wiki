# Installation Guide

## Mounting Location Requirements

- **Install at a central location** to ensure audibility in all rooms
- Corridor, living room ceiling, or high wall
- At least **2 m above floor** (prevent accidental contact)
- When used with smoke/CO sensors, install in bedroom hallway

## Installation Steps

1. Select mounting location (recommended: hallway or living room)
2. Secure mounting base to wall or ceiling with screws
3. Connect power (mains version) or insert battery (battery version)
4. Install alarm body onto base
5. Pair via gateway
6.linkage test: trigger any sensor, siren should respond

## Important Notes

- Install at a height unreachable by children
- Periodically test alarm volume and lights for proper operation
- For battery version, replace batteries when low
- Can set linkage rules with other sensors
