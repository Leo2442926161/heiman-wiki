# Installation Guide

## Wearing Methods

- **Belt clip**: clip onto belt or waistband
- **Lanyard**: hang around neck, resting on chest

## Usage Instructions

1. Insert battery
2. Choose wearing method (belt clip or lanyard)
3. Wear on body
4. Pair via gateway
5. Simulation test:
   - From standing, quickly crouch down or lie on floor
   - Sensor should detect fall and trigger alarm within seconds
   - PressSOS button to test manual alarm

## Important Notes

- Recommended to **wear all day** (can be removed for charging/battery change during shower)
- Uses AI algorithm to distinguish falls from daily activities (sitting, bending), low false alarm rate
- Auto-alarm on fall detection, also supports manual buttonSOS
- Can linkage with emergency button for dual protection
