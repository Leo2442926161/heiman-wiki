# Certification Standards

The HS1SA-Z has passed the following certifications:

- **EN 14604** — European standard for smoke alarms, see [EN European Standards](../../../../standard/EN-European-Standards.md)
- **Z-Wave S2 Certification** — Supports S2 security framework, see [Other Certifications](../../../standard/Other-Certifications.md)
- **CE** — EU Declaration of Conformity
