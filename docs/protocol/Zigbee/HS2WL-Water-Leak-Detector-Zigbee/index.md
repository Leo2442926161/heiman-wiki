# HS2WL - Water Leak Detector (Zigbee)

## Overview

The HS2WL is Heiman's second generation Zigbee water leak detector, supporting dual-channel detection via probe and cable, suitable for detecting water leak events such as pipe bursts and sink overflows.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Detection Method | Probe Detection + Cable Detection (Dual Channel) |
| Power Supply | DC 3V Battery |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH |

## Features

- Real-time water leak / accumulation detection
- Probe + cable dual-channel detection
- Buzzer alarm
- Low battery reminder
- Probe anti-corrosion design

## Compatible Accessories

- Extension detection cable
