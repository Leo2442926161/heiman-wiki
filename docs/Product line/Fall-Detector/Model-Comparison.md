# Model Comparison

## Model Comparison

| Model | Protocol | Sensor | Power | Features | Usage Suggestions |
|------|------|--------|------|------|----------|
| [HS2FD](../../无线协议/Zigbee/HS2FD-Fall-Detector-Zigbee) | Zigbee | Accelerometer + Gyroscope | Battery | AI algorithm fall detection | Wearable design, auto fall detection + one-clickSOS, suitable for elderly living alone and nursing homes |

## Scenario Recommendations

| Scenario | Recommended Model | Automation Scheme |
|------|----------|----------|
| Elderly living alone monitoring | HS2FD | Fall detected → auto push to children's phone → no response timeout auto-call 120 |
| Nursing homes | HS2FD | Centralized management platform + floor siren alarm + caregiver handheld simultaneous alert |
| Post-surgery rehabilitation | HS2FD | Patient gets out of bed with fall risk detected → caregiver station early warning |
| Bathroom/nighttime | HS2FD | IP rated design, can wear while showering + nighttime fall auto-alert |

## Selection Guide

- **Home elderly care**: [HS2FD](../../无线协议/Zigbee/HS2FD-Fall-Detector-Zigbee) (wearable, AI detection)
- **Pair with emergency button**: HS2FD auto fall detection + emergency button manualSOS, dual protection
