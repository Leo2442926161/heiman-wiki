# Installation Guide

## Mounting Location Requirements

- **Mount at breathing height** — approximately **1.5 m** above floor
- Near bedroom areas to ensure alarm can be heard during sleep
- **Do not install in corners**; maintain air circulation
- Keep away from vents, windows, fans and other airflow-affecting locations

## Installation Steps

1. Determine mounting location (recommended: bedroom hallway or living room)
2. Fix mounting base to the wall (screws or 3M adhesive)
3. Insert battery, sensor powers on automatically
4. Install the main unit onto the base
5. Self-test: press the test button, alarm LED flashes and alarm sounds
6. Pair via gateway

## Important Notes

- Sensor lifespan typically **7 years**, replace entire unit when expired
- Test functionality once a month
- Do not block the sensor ventilation holes
- If alarm sounds, first ventilate the room by opening windows, then check for CO source leaks
