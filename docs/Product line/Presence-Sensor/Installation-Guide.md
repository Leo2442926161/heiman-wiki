# Installation Guide

## Mounting Location Requirements

- **Ceiling mount**, sensor pointing vertically downward
- Mounting height **2.5 ~ 3.5 m**
- Detection radius approximately **3 ~ 5 m**
- **Avoid**: metal obstacles in front of the sensor (metal blocks millimeter waves)
- Keep away from large metal pipes, AC indoor units, etc.

## Installation Steps

1. Select ceiling mounting location (center of room is best)
2. Drill holes in ceiling, secure base with expansion screws
3. Connect wiring (if mains-powered) or insert battery
4. Rotate and snap sensor body into base
5. After power-on, indicator lights up, sensor begins self-test
6. Open APP to pair and add device
7. Test: enter detection area and sit still — sensor should maintain "occupied" status

## Important Notes

- Millimeter waves can penetrate thin panels (drywall, wood), but metal blocks them
- Thick walls, metal ceilings will reduce detection range
- Recommended to use with PIR sensor for full scene coverage
