# PIR Sensor

The PIR (Passive Infrared) Sensor detects movement by sensing infrared heat radiation emitted by the human body, and is the most commonly used presence detection solution in smart homes.

## Product Series

| Series | Communication | Features |
|------|----------|------|
| HS1MS | Zigbee | Basic model, 110° detection angle |
| HS3MS | Zigbee | Upgraded model, 360° ceiling mount |
| M1 Series | Matter / Zigbee | Supports Matter over Thread, selectable 110° or 360° |
| HS8MLS | Zigbee | Long range model |
| HS8MIS | Zigbee | Combined PIR + light sensor |

## Core Features

- Motion detection
- Light level detection (M1/HS8MIS series)
- Detection distance ≤7m (25°C)
- Mounting: wall or ceiling
- Battery life: 3 years (Zigbee) / 1 year (Matter)

## Technical Specifications (M1 Series)

- Detection angle: 110° / 360°
- Detection distance: ≤7m
- Illuminance range: 0~1200 Lux
- Power: DC 3V (Lithium battery)
- Battery life: 3 years (Zigbee) / 1 year (Matter)

## Scenario Examples

- Light on when motion detected, off when vacant
- Security: abnormal motion detection triggers alarm notification
- Away arming: auto-turn off AC/heating when unoccupied for energy saving
