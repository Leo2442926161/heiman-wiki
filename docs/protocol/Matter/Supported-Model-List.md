# Supported Model List

All Heiman Matter devices use the Matter over Thread solution. The following are the released supported models.

## Security Alarm Category

| Model | Product Name | Description |
|------|----------|------|
| HS1SA-M | Smart Smoke Alarm (Matter) | US/EU Version, world's first Matter-certified smoke alarm |
| HS1CA-M | Smart Carbon Monoxide Alarm (Matter) | US/EU Version, world's first Matter-certified CO alarm |

## Sensor Category

| Model | Product Name | Description |
|------|----------|------|
| M1-M | PIR Sensor (Matter) | Passive infrared detection, supports Apple HomeKit, Google Home, etc. |
| L1-M | Water Leak Detector (Matter) | Water leak detection, real-time alert push |
| H1-M | Temperature and Humidity Sensor (Matter) | Temperature and humidity detection, environmental monitoring |
| HS1DS-M | Door/Window Sensor (Matter) | Door/window open/close detection, tamper alarm |

## Notes

- Matter devices require a **Thread Border Router** to use, such as Apple HomePod mini, Google Nest Hub 2nd Gen, etc.
- The current Heiman Matter product line is still expanding, with more models to be released gradually.
