# M1-Z - PIR Motion Sensor (Zigbee)

## Overview

The M1-Z is Heiman's compact Zigbee PIR motion sensor, supporting 110° or 360° detection angle (depending on installation method), with a built-in illuminance sensor, suitable for smart lighting and security applications.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Passive Infrared (PIR) + Illuminance |
| Detection Angle | 110° (wall mount) / 360° (ceiling mount) |
| Detection Distance | ≤7 m (25°C) |
| Illuminance Range | 0 ~ 1200 Lux |
| Power Supply | DC 3V Lithium Battery |
| Battery Life | Approx. 3 years |
| Operating Temperature | -10°C ~ +50°C |
| Installation Method | Wall Mount / Ceiling Mount |

## Features

- Motion detection
- Illuminance detection (0~1200 Lux)
- 110° / 360° switchable
- Low battery reminder
- Tamper alarm
- Compact design
