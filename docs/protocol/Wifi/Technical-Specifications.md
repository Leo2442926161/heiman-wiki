# Technical Specifications

The following are the general technical specifications for HEIMAN Wi-Fi devices.

## Wireless Parameters

| Parameter | Value |
|------|-----|
| Protocol Standard | IEEE 802.11 b/g/n |
| Operating Band | 2.4 GHz (not compatible with 5 GHz) |
| Data Rate | Up to 150 Mbps (802.11n) |
| Network Topology | Star topology (direct connection to router) |
| Communication Range | Indoor approx. 30~50 m (depends on router performance) |
| Encryption | WPA2-PSK / WPA3 |

## Pairing Methods

| Method | Description |
|------|------|
| SmartConfig | One-touch pairing via mobile app (traditional Wi-Fi devices) |
| Bluetooth-assisted pairing | Configure Wi-Fi credentials via Bluetooth temporary channel (some new models) |
| AP Mode | Device creates a hotspot, configure by connecting via phone |

## Electrical & Power

| Parameter | Value |
|------|-----|
| Power Supply | Mains powered (plug/gateway) or battery powered (alarm) |
| Power Consumption | Relatively higher than Zigbee / Thread due to maintaining TCP/IP connection |
| Standby Power | Plug type approx. 0.5W ~ 1W |

## Environmental Parameters

| Parameter | Value |
|------|-----|
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤ 90% RH (non-condensing) |
