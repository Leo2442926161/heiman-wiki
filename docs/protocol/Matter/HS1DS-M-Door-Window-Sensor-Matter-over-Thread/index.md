# HS1DS-M - Smart Door/Window Sensor (Matter over Thread)

## Overview

HS1DS-M is the Matter over Thread version of Heiman's smart door/window sensor, supporting native integration with Apple Home, Google Home, Amazon Alexa, Home Assistant, and all Matter-compatible platforms without the need for a proprietary gateway.

## Technical Specifications

| Parameter | Value |
|------|-----|
| Communication Protocol | Matter over Thread |
| Power Supply | CR1632 Coin Cell Battery |
| Battery Life | Approx. 2-3 years |
| Installation Gap | ≤10 mm |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤95% RH |

## Features

- Real-time door/window open/close status detection
- Tamper alarm
- Low battery reminder
- Cross-platform compatibility: Apple Home / Google Home / Alexa / SmartThings / Home Assistant
- Thread Mesh network, low-latency local control
- No dedicated gateway required, requires Thread Border Router
