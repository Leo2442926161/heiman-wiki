# Certification Standards

HS1SA-E has obtained the following certifications:

- **EN 14604** — European standard for smoke alarms, see [EN European Standards](../../../../standard/EN-European-Standards.md)
- **CE / RED / EMC** — EU radio equipment directive and electromagnetic compatibility
- **Zigbee 3.0 / Q-Label** — Zigbee Alliance certification, see [Other Certifications](../../../standard/Other-Certifications.md)
- **Red Dot Design Award** — Red Dot product design award
