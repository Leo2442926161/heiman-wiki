# HS2SK - Smart Plug (Zigbee)

## Overview

The HS2SK is Heiman's Zigbee smart metering plug, supporting remote on/off control and real-time power metering with ±2% accuracy, suitable for smart management of household appliances.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Rated Voltage | AC 220V (China standard) / AC 110V (US standard) |
| Max Load | 16A / 3680W |
| Metering Accuracy | ±2% |
| Power Supply | Mains Powered |

## Features

- Remote on/off control
- Real-time power / energy metering
- Overload protection
- Scheduled on/off
- Energy statistics
- Supports scene automation

## Application Scenarios

- Remotely turn off forgotten appliances
- Track air conditioner / water heater energy consumption
- Away mode: one-tap shut off non-essential appliances
