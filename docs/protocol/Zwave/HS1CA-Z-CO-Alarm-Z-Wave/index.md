# HS1CA-Z Smart Carbon Monoxide Alarm (Z-Wave)

The HS1CA-Z is a **Z-Wave** smart carbon monoxide alarm launched by Heiman, using electrochemical sensor technology, compatible with the Z-Wave Plus standard.

## Product Overview

- **Product Type**: Smart Carbon Monoxide Alarm
- **Communication Protocol**: Z-Wave Plus
- **Certification Standard**: EN 50291
- **Typical Scenarios**: CO leak warning in homes, apartments, etc.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Sensor Type | Electrochemical CO sensor |
| Communication Protocol | Z-Wave Plus (ITU-T G.9959) |
| Operating Frequency | 868.42 MHz (EU) / 908.42 MHz (US) |
| Detection Range | 0~1000 ppm |
| Alarm Sound Level | ≥85 dB (at 3 m) |
| Power Supply | Battery powered |
| Sensor Life | Typical 7 years |
| Encryption Method | S2 Authenticated / S2 Unauthenticated |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤90% RH (non-condensing) |

## Features

- High-precision electrochemical CO sensor
- Concentration level-based alarm
- 85 dB high sound and light alarm
- Low battery reminder
- Sensor end-of-life reminder
- Self-test function
- Z-Wave S2 security encryption
- Supports OTA firmware update

## Certifications

- EN 50291 certified
- Z-Wave Plus certified
- S2 security certified
