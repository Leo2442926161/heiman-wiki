# JT Series - Gas Emergency Shut-off Valve (Zigbee)

## Overview

The JT series gas emergency shut-off valve is an actuation device that works with Heiman Zigbee combustible gas detectors. When a gas leak is detected, it can remotely or automatically close the gas pipeline valve to prevent accidents.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Type | Standalone + Zigbee Linked |
| Detected Gas | Natural Gas / Liquefied Petroleum Gas |
| Certification Standard | GB 15322 |
| Operating Temperature | -10°C ~ +50°C |

## Features

- Automatic valve closure in linkage with HS1CG / HS3CG
- Supports remote manual valve closure
- Leak alarm push notification
- Solenoid valve actuation

## Certification Standards

- GB 15322

## Compatible Accessories

- HS1CG / HS3CG Combustible Gas Detector
