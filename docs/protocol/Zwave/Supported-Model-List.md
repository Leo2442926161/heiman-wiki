# Supported Model List

Heiman Z-Wave products are mainly aimed at the European and American smart home markets and are compatible with the Z-Wave Plus standard.

## Security & Alarms

| Model | Product Name | Description |
|-------|--------------|-------------|
| HS1SA-Z | Smart Smoke Alarm (Z-Wave) | Z-Wave version smoke alarm |
| HS1CA-Z | Smart Carbon Monoxide Alarm (Z-Wave) | Z-Wave version CO alarm |
| HS1CG-Z | Combustible Gas Detector (Z-Wave) | Z-Wave version gas detector |

## Sensors

| Model | Product Name | Description |
|-------|--------------|-------------|
| HS1DS-Z | Door Sensor (Z-Wave) | Z-Wave version door/window sensor |
| HS1MS-Z | PIR Motion Sensor (Z-Wave) | Z-Wave version motion detection |
| HS1WL-Z | Water Leak Detector (Z-Wave) | Z-Wave version water leak detection |

## Compatibility

| Item | Description |
|------|-------------|
| Standard | Z-Wave Plus (500 / 700 series chips) |
| Compatible Hubs | Homey, Hubitat, SmartThings, Vera, Fibaro, Home Assistant (Z-Wave JS) |
| S2 Security | Supports S2 security protocol (Unencrypted / S2 Unauthenticated / S2 Authenticated) |
| SmartStart | Supported (700 series chip models) |
| OTA Update | Supports FW Update CC |
