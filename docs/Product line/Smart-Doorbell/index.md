# Smart Doorbell

The Heiman Smart Doorbell integrates doorbell calling and security monitoring functions, supporting remote video intercom.

## Core Features

- Doorbell button trigger
- Remote real-time video (Wi-Fi version)
- Two-way audio
- Motion detection recording
- Night vision

## Applicable Scenarios

- Visitor call: answer whether at home or away
- Delivery/takeout notification: remotely confirm and open door
- Security monitoring: record suspicious activity at the door

> **Note**: Heiman doorbell product line is primarily Zigbee / Wi-Fi based. Please consult official sources for specific model specifications.
