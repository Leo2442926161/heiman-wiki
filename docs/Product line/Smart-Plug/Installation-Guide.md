# Installation Guide

## Mounting Location Requirements

- Plug directly into wall outlet, **do not daisy-chain** (do not insert into power strip or extension cord)
- Ensure outlet rated current ≥ 16A
- Do not install in damp or high-temperature environments

## Installation Steps

1. Plug smart plug directly into 220V (or 110V) wall outlet
2. LED lights up when powered on
3. Press and hold the switch button to enter pairing mode
4. Add device in Heiman APP
5. After successful pairing, remote on/off control is available

## Safety Precautions

- **Max load**: 16A / 3680W (EU standard), 15A / 1800W (US standard)
- **Do not overload**: check power limits when using high-power appliances (AC, heater)
- **Do not daisy-chain**:prohibited inserting smart plug into power strip
- Do not cover or block the plug housing
- Periodically check if the plug is overheating

## Typical Applications

- Timer control for low-power devices: water heater, humidifier, aquarium light, etc.
- Sensor: light on when motion detected, off when vacant
- Energy tracking: monitor appliance energy consumption
