# Installation Guide

## Mounting Location Requirements

- Install at **1.2 ~ 1.5 m** height (representative of ambient average)
- **Keep away from**: direct sunlight, doors/windows, AC vents, radiators
- **Keep away from**: humidifiers, kitchen stoves, and other local temperature/humidity sources
- Choose frequently occupied rooms (bedroom, living room, nursery)

## Installation Steps

1. Select mounting location (wall or tabletop both OK)
2. Wall mount: secure base with screws or 3M adhesive
3. Insert CR2 battery (note polarity)
4. Sensor powers on automatically, display (if available) shows temp/humidity
5. Pair via gateway
6. Check data in APP to confirm normal operation

## Important Notes

- When wall-mounted, ensure sensor ventilation holes are not blocked
- Battery life ~2-3 years
- Different rooms' temperature/humidity data can be collected via multiple sensors
- Can be used to linkage AC, dehumidifier, humidifier, etc.
