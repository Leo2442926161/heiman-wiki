# Smart Plug

The Heiman Smart Metering Plug supports remote on/off control and real-time power consumption monitoring, helping users achieve intelligent appliance management.

## Product Series

| Series | Communication | Features |
|------|----------|------|
| HS2SK Series | Zigbee | Metering type, supports power monitoring |

## Core Features

- Remote on/off control
- Real-time power/energy monitoring
- Overload protection
- Timer on/off
- Energy statistics

## Technical Specifications

- Rated voltage: AC 220V (China standard) / AC 110V (US standard)
- Max load: 16A / 3680W
- Metering accuracy: ±2%

## Scenario Examples

- Remotely turn off appliances left on
- Track AC/water heater energy consumption
-linkage away mode: one-click turn off all non-essential appliances
- Timer control for aquarium lights, humidifiers, etc.
