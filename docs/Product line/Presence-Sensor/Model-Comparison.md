# Model Comparison

## Model Comparison

| Model | Protocol | Detection Technology | Mounting | Features | Usage Suggestions |
|------|------|----------|------|------|----------|
| [HS8OS](../../无线协议/Zigbee/HS8OS-Presence-Sensor-Zigbee) | Zigbee | Millimeter-wave radar | Ceiling | Stationary presence detection | More precise than PIR, ideal for bathroom/bedroom where people stay still for long periods, solves PIR light-off issue when stationary |

## Scenario Recommendations

| Scenario | Recommended Model | Automation Scheme |
|------|----------|----------|
| Bathroom always-on light | HS8OS | Lights on when occupied, off when vacant (detects micro-breathing, solves PIR stationary light-off pain point) |
| Bedroom sleep monitoring | HS8OS | Detect if someone is in bed,linkage HVAC to auto-adjust temperature |
| Elderly care | HS8OS | Prolonged stillness in bathroom → system alerts family, preventing accidents |
| Meeting room energy saving | HS8OS | Unoccupied for N minutes → auto turn off lights and AC, save energy |

## Selection Guide

- **Solve PIR light-off pain point**: [HS8OS](../../无线协议/Zigbee/HS8OS-Presence-Sensor-Zigbee) (millimeter-wave, stationary detection)
- **Pair with PIR**: PIR detects movement + HS8OS detects stationary presence, complementary use for best results
