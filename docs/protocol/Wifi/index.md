# Wi-Fi

Wi-Fi is the most ubiquitous wireless networking protocol, based on the IEEE 802.11 standard. Some HEIMAN smart devices support Wi-Fi connectivity and can connect directly to a router without needing an additional gateway.

## Applicable Scenarios

Wi-Fi devices are suitable for:

- Environments with established network infrastructure, no need for an extra gateway
- Devices with low power sensitivity (e.g., smart plugs)
- Scenarios requiring remote access and cloud services
- Homes with mature Wi-Fi Mesh network coverage

## HEIMAN Wi-Fi Products

HEIMAN offers products supporting Wi-Fi connectivity:

- **Smart Plug HS2SK-WiFi** — Power metering, remote on/off
- **Smart Gateway** — Can connect to router via Wi-Fi

## Notes

- Wi-Fi sensor battery consumption is higher than Zigbee/Thread
- A large number of Wi-Fi devices may affect network stability
- HEIMAN's main product line focuses on Zigbee / Matter over Thread; the Wi-Fi product line is relatively streamlined
