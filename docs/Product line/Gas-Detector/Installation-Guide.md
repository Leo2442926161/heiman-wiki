# Installation Guide

## Mounting Location Requirements

- **Natural gas (methane)**: lighter than air, install **within 30 cm** of ceiling
- **Liquefied petroleum gas (propane)**: heavier than air, install **within 30 cm** of floor
- Near potential leak points such as gas stoves, water heaters, gas pipe joints

## Installation Steps

1. Confirm gas type to be detected (natural gas or LPG)
2. Select corresponding mounting height
3. Secure mounting base to wall with screws
4. Connect power (most models are mains-powered)
5. Power on sensor, allow ~3-5 minutes warm-up
6. Self-test: press test button, verify audible and visual alarm
7. Pair via gateway (Zigbee version)
8. If linkage solenoid valve, connect shutoff valve control wire

## Important Notes

- Periodically test sensor sensitivity with test gas
- Sensor lifespan ~3-5 years, replace when expired
- If you smell gas, do NOT operate electrical switches or use open flames
- Mounting height depends on gas type (natural gas high, LPG low)
