# HS1SA-Z Smart Smoke Alarm (Z-Wave)

The HS1SA-Z is a **Z-Wave** smart smoke alarm launched by Heiman, using photoelectric smoke detection technology, compatible with the Z-Wave Plus standard, designed for the European and American smart home markets.

## Product Overview

- **Product Type**: Smart Smoke Alarm
- **Communication Protocol**: Z-Wave Plus
- **Certification Standard**: EN 14604
- **Typical Scenarios**: Fire warning in homes, apartments, etc.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Sensor Type | Infrared photoelectric |
| Communication Protocol | Z-Wave Plus (ITU-T G.9959) |
| Operating Frequency | 868.42 MHz (EU) / 908.42 MHz (US) |
| Alarm Sound Level | ≥85 dB (at 3 m) |
| Power Supply | CR123A lithium battery |
| Battery Life | Approx. 2~3 years |
| Encryption Method | S2 Authenticated / S2 Unauthenticated |
| Operating Temperature | -10°C ~ +50°C |
| Operating Humidity | ≤90% RH (non-condensing) |

## Features

- Photoelectric smoke detection, capable of detecting smoldering and open flame smoke
- 85 dB high sound and light alarm
- Low battery reminder
- Self-test function
- Tamper alarm
- Z-Wave S2 security encryption
- Supports OTA firmware update

## Certifications

- EN 14604 certified
- Z-Wave Plus certified
- S2 security certified
