# HS3MS - PIR Motion Sensor (Zigbee)

## Overview

The HS3MS is Heiman's 3rd generation Zigbee PIR motion sensor, featuring 360° ceiling-mount design with significantly improved detection sensitivity and coverage over the previous generation.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Passive Infrared (PIR) |
| Detection Angle | 360° (ceiling mount) |
| Detection Distance | ≤7 m (25°C) |
| Power Supply | DC 3V Lithium Battery |
| Battery Life | Approx. 3 years |
| Operating Temperature | -10°C ~ +50°C |
| Installation Method | Ceiling Mount |

## Features

- Motion detection
- 360° omnidirectional coverage
- Low battery reminder
- Tamper alarm
- Improved detection sensitivity
- Supports scene automation
