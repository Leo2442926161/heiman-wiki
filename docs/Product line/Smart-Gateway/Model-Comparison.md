# Model Comparison

## Model Comparison

| Model | Zigbee Sub-device Limit | Network | Features | Usage Suggestions |
|------|------------------|----------|------|----------|
| [HS6GW](../../无线协议/Zigbee/HS6GW-Smart-Gateway-Zigbee) | 128 | WiFi + Ethernet | Latest model, supports future Thread upgrade | Heiman ecosystem core, remote management via Heiman APP, ideal for hassle-free users |
| HS5GW | 64 | WiFi + Ethernet | High performance | Medium-large projects, stable and reliable |
| HS3GW | 32 | WiFi | Enhanced | Adequate for small-medium homes |
| HS1GW | 16 | WiFi | Basic | Entry-level, suitable for few devices |

## Scenario Recommendations

| Scenario | Recommended Model | Description |
|------|----------|------|
| Whole-home Heiman ecosystem | HS6GW | Use with Heiman Zigbee sensors, remote management via Heiman APP, no server setup needed |
| Third-party platform users | — | Recommended to buy Zigbee coordinator (e.g. SLZB-06) with Zigbee2MQTT / ZHA for more flexibility |
| Small-medium home | HS3GW | 32 device limit, meets basic needs |

## Selection Guide

- **Heiman own ecosystem**: [HS6GW](../../无线协议/Zigbee/HS6GW-Smart-Gateway-Zigbee) (128 devices, WiFi + Ethernet)
- **Third-party platform**: Recommended universal Zigbee coordinator + Zigbee2MQTT / ZHA
