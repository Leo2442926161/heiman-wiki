# HS8MIS - Motion Sensor (Zigbee)

## Overview

The HS8MIS is Heiman's multi-function Zigbee motion sensor, integrating PIR human detection and illuminance detection, suitable for smart scenarios requiring both light and motion monitoring.

## Technical Specifications

| Parameter | Value |
|-----------|-------|
| Communication Protocol | Zigbee 3.0 |
| Sensor Type | Passive Infrared (PIR) + Illuminance |
| Detection Angle | 110° |
| Detection Distance | ≤7 m |
| Power Supply | DC 3V Battery |

## Features

- Motion detection
- Illuminance detection
- Low power design
- Supports scene automation
